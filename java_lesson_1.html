Java Programming - Lesson 1

Introduction to Java

Java is a high-level, class-based, and object-oriented programming language designed to have as few implementation dependencies as possible. It is one of the most widely used programming languages, known for its portability, versatility, and extensive community support.

1. Why Learn Java?

Java is a popular choice for building large-scale enterprise applications, Android apps, web servers, and many more. Key reasons to learn Java include:

  • Platform Independence: Java programs can run on any platform with a compatible JVM (Java Virtual Machine).
  • Object-Oriented: Helps to structure complex programs into reusable pieces of code (objects).
  • Robust and Secure: Java has strong memory management and security features.

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");  // Outputs "Hello, World!" to the console
    }
}

Explanation of the structure:

  • public class HelloWorld: Declares a class named HelloWorld. In Java, every application must have at least one class.
  • public static void main(String[] args): This is the main method from where Java application execution begins.
  • System.out.println("Hello, World!"); Prints the text "Hello, World!" to the console.

3. Setting Up Java Development Environment

To write and run Java programs, you need to set up the Java Development Kit (JDK) and an Integrated Development Environment (IDE) like IntelliJ IDEA or Eclipse:

  • Download and install the latest version of the JDK from the official Oracle website.
  • Choose and install an IDE of your preference, such as IntelliJ IDEA, Eclipse, or NetBeans.
  • Configure the IDE to recognize the installed JDK and start coding.

4. Compiling and Running a Java Program

After writing a Java program, you can compile and run it using the following commands in the command line:

javac HelloWorld.java  // Compiles the Java file
java HelloWorld        // Runs the compiled Java class

If there are no syntax errors, the program will execute and display "Hello, World!" in the console.
