Lesson 2: HTML Elements and Attributes

1. Understanding HTML Elements

HTML elements are the building blocks of an HTML page. Each element typically consists of a start tag, content, and an end tag. Let’s look at the basic structure of an HTML element:

<tagname>Content goes here</tagname>

The start tag includes the element’s name, enclosed in angle brackets. The end tag also includes the element’s name but is prefixed with a forward slash (/).

1.1. Example: Basic HTML Element

<p>This is a paragraph element.</p>

2. Types of HTML Elements

HTML elements can be classified into two categories:

  • Block-level elements: They occupy the entire width available and start on a new line. Examples include <div>, <h1> to <h6>, and <p>.
  • Inline elements: They occupy only as much width as necessary and do not start on a new line. Examples include <span>, <a>, and <img>.

2.1. Example: Block-level vs. Inline Elements

In the following example, the <div> is a block-level element, and the <span> is an inline element.

<div>
    This is a block-level element.
</div>
<span>This is an inline element.</span>

3. Understanding HTML Attributes

HTML attributes provide additional information about HTML elements. They are specified within the start tag of an element and usually come in name/value pairs. Here’s an example:

<tagname attribute="value">Content</tagname>

Attributes are used to configure elements or modify their behavior. For example, the href attribute in an <a> (anchor) element specifies the URL to which the link points.

3.1. Common HTML Attributes

  • id: A unique identifier for the element.
  • class: Specifies one or more class names for an element (used for CSS styling).
  • style: Allows you to add inline CSS styles to an element.
  • title: Provides additional information about an element (visible as a tooltip).
  • src: Specifies the URL of an image, video, or audio file.
  • alt: Specifies alternative text for an image, used for accessibility.

4. Practical Examples

Let’s look at some practical examples of using HTML elements with attributes.

4.1. Example 1: Using <a> Tag with Attributes

The <a> tag defines a hyperlink. It uses the href attribute to specify the destination URL and the title attribute to provide additional information:

<a href="https://www.example.com" title="Visit Example">Click here to visit Example</a>

Output: Click here to visit Example

4.2. Example 2: Using <img> Tag with Attributes

The <img> tag is used to display an image. It uses the src attribute to specify the image source and the alt attribute to provide alternative text:

5. Advanced Attributes

Some advanced attributes allow you to control the behavior of elements:

  • target: Specifies where to open the linked document (e.g., _blank for a new tab).
  • data-*: Custom data attributes for storing extra information.
  • rel: Specifies the relationship between the current document and the linked document.
